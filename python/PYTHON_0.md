> ### 튜플 

* 튜플은 수정이 불가능한 시퀀스로 인덱스로 접근
* 소괄호 () 혹은 tuple()을 통해 생성
* 값에 대 한 접근은 tuple[i]




> ### 레인지(range)

* range는 숫자의 시퀀스를 나타내기 위해 사용
* 기본형 range(n) -> 0부터 n-1까지 숫자의 시퀀스
* 범위 지정 : range(n,m) 
  * n부터 m-1까지 숫자의 시퀀스
* 범위 및 스텝 지정 : range(n,m,s)
  * n부터 m-1까지 s만큼 증가

> ### 이스케이프 시퀀스

## 제어문(Control Statement)

>* 파이썬은 기본적으로 위에서부터 아래로 순차적 명령수행
>
>* 특정 상황에 따라 코드를 선택적으로 실행하는 제어가 필요



### 1.조건문

```python
if <조건>:    # 참일경우
    <조건 실행>
else:        # 거짓일경우
 	<조건실행>	
```



```py
#예제

a = 5
if a > 5:
	print('5 초과')
else:
	print('5 이하')
print(a)
```



```python
#예제2 : 사용자에게 입력받아 홀수인지 짝수인지 출력하시오.

if a % 2 == 0
```

## 반복문

> * while 문 : 종료 조건에 해당하는 코드를 통해 반복문을 종료시켜야 함
> * for 문  : 반복가능한 객체를 모두 순회하면 종료
> * 반복 제어 ( break, continue, for-else)



### while문

* 조건식이 참인경우 반복적으로 코드실행

```py
a = 0

while a < 5:
	print(a)
    a=a+1
    
print ('끝')

```



### 반복문제어 Break

* 특정 조건 실행시 반복문 종료



### 반복문제어 continue

* continue 이후 코드 블록은 수행되지 않고 다음 반복을 실행

```python

    if i % 2 == 0:
         continue
    print(i)
    
    ##홀수만 출력되는 코드 ##
```
